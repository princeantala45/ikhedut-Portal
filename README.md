# 🌱 About Project

The **iKhedut Portal** is a web platform designed to simplify access to agriculture schemes and services for farmers.

The portal allows farmers to easily explore available schemes, view information, and interact with a user-friendly digital system.  
It demonstrates how technology can help improve accessibility of government services in the agriculture sector.
